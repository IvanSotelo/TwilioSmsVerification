## Introduction

TwilioVerify makes it easier and safer than custom verification systems to verify a userâ€™s phone number. It ensures that the phone number is valid by sending a short code via SMS to the number during registration. This can help reduce the number of fake accounts created and failure rates when sending SMS notifications to users.
